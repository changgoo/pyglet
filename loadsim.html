load_athdf(num=None, output_id=None, load_method='xarray')[source]#

Read Athena hdf5 file and convert it to xarray Dataset

Parameters

  • num (int) – Snapshot number, e.g., /basedir/problem_id.00042.athdf

  • output_id (int (optional)) – Output id given in the <output#> block. Useful when there are more than one athdf outputs.

  • load_method (str (optional)) – Method to read athdf file. Available options are ‘xarray’ or ‘yt’. Default is ‘xarray’
